# Knowledge Pack — migration régionale

Cette note accompagne la migration des profils métier tunisiens vers `knowledge/ports/`.

Ports migrés : Gammarth, Sidi Bou Saïd, Ghar El Melh, Ras Fartass, El Haouaria et Kélibia.

Pantelleria reste volontairement exclue du profil métier jusqu’à validation offshore dédiée.

Les champs `zones` restent vides jusqu’à validation terrain, cartographique et nautique.
