# Ras Fartass — destination côtière de référence

## Politique de destination

Pour FABLE, **Ras Fartass** est la destination côtière du cap Bon utilisée pour les sorties familiales au départ de Gammarth. Elle est également couramment fréquentée par les plaisanciers venant de Sidi Bou Saïd.

**Korbous est un lieu distinct.** Il ne doit jamais être utilisé comme alias, point météo de remplacement ou libellé de Ras Fartass.
